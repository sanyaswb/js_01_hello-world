## Задание
- Написать функцию <b>helloWorld</b>, которая возвращает строку 'Hello, World'.

## Примечание
- Рабочий файл: index.js.
- Функция должна вернуть строку, а не вывести её в консоль.

## Примеры
- () => 'Hello, World'
